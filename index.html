Projets

ARES/DEMETER

ARES/DEMETER sont deux projets de jeu de plateau développé en équipe. Chaque jeu est constitué d’un plateau divisé en tuiles représentant des ressources (forêts, montagnes, pâturages, champs, mer), avec des bâtiments pouvant être déployés. Les joueurs collectent des ressources et réalisent des actions stratégiques à chaque tour. Le projet est conçu pour être extensible afin d'ajouter facilement de nouvelles fonctionnalités.

Technologies : Java, Git, Modélisation objet

Fonctionnalités :

  • Modélisation du plateau et des ressources
  • Gestion des bâtiments et de leur dimension
  • Système de collecte et d’échange de ressources
  • Déroulement du jeu basé sur des tours et actions limitées
  • Extensible pour intégrer de nouvelles règles ou îles
Voir le projet sur GitHub

StarShip Game

Jeu vidéo simple où le joueur contrôle un vaisseau verticalement et doit détruire des soucoupes volantes avant qu'elles ne passent.

Technologies : JavaScript, HTML5, Canvas

Fonctionnalités :

  • Déplacement du vaisseau avec le clavier
  • Soucis volantes aléatoires à détruire
  • Gestion du score
Voir le projet sur GitHub

Task Manager App

Application de gestion de tâches avec attribution de priorité. Les tâches peuvent être distinguées entre celles à réaliser et celles déjà terminées.

Technologies : React.js, JavaScript, CSS

Fonctionnalités :

  • Affichage des tâches par ordre de priorité décroissante
  • Ajout de nouvelles tâches avec priorité personnalisée
  • Gestion des tâches terminées et à réaliser
  • Optionnelle : sauvegarde locale avec LocalStorage
